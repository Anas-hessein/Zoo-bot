## About

Zoo-bot is a slack bot. It has a 8 different slash commands about animls. Built only with VS code and slack api site. Each command hae there own discreaption. 

## Animls 
dogs, cats, ducks, and foxs.

## commands
| Name | Description |
|------|-------------| 
| /zoo-ping | a ping from my nice bot |
| /zoo-joke | a nice joke from my nice bot |
| /zoo-catfact | a cat fact from my nice bot |
| /zoo-dogfact | a dog fact from my nice bot |
| /zoo-catpic |	a cat pic generated by Zoo |
| /zoo-dogpic | a dog pic generate command from Zoo |
| /zoo-duckpic | a duck pic that generated with the power of zoo |
| /zoo-foxpic | a fox pic |
